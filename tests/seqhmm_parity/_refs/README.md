# MNHMM seqHMM Golden Fixtures

This directory stores cached R `seqHMM` reference outputs for the MNHMM
parity tests in `tests/seqhmm_parity/test_mnhmm_matrix.py`.

Each scenario is stored in its own subdirectory. The `r_reference_mnhmm_*.R`
scripts beside this folder show how the cached CSV references were generated.
