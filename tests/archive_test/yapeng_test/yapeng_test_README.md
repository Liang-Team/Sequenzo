# Dissimilarity Measure Performance Optimization

Performance benchmarking and optimization of Sequenzo's dissimilarity measure algorithms (OM, LCP, LCS, EUCLID) against TanaT.

## Directory Structure

```
yapeng_test/
├── README.md
├── generated_datasets/          # Test datasets (generated by generation.py)
│   ├── output_n10000_l10_u85/   # Exp1: sequence length
│   ├── output_n10000_l30_u85/   # Shared across all 3 experiments
│   ├── output_n10000_l50_u85/
│   ├── output_n10000_l100_u85/
│   ├── output_n10000_l30_u20/   # Exp2: uniqueness rate
│   ├── output_n10000_l30_u50/
│   ├── output_n10000_l30_u100/
│   ├── output_n1000_l30_u85/    # Exp3: sample size
│   ├── output_n2000_l30_u85/
│   ├── output_n3000_l30_u85/
│   └── output_n4000_l30_u85/
├── run_sequenzo.py              # OM benchmark (Sequenzo)
├── run_tanat.py                 # OM benchmark (TanaT)
├── run_sequenzo_other.py        # LCP/LCS/EUCLID benchmark (Sequenzo)
├── run_tanat_other.py           # LCP/LCS/EUCLID benchmark (TanaT)
└── plot_runtime/                # Visualization scripts
    ├── om_baseline_benchmark.py
    └── other_baseline_benchmark.py
```

## Experiment Design

Three controlled experiments, each varying one parameter while fixing the other two:

| Experiment | Variable | Values | Fixed |
|------------|----------|--------|-------|
| Exp1: Sequence Length | L | 10, 30, 50, 100 | n=10000, U=85 |
| Exp2: Uniqueness Rate | U | 20, 50, 85, 100 | n=10000, L=30 |
| Exp3: Sample Size | n | 1k, 2k, 3k, 4k, 10k | L=30, U=85 |

Data generated using `random-sequences-generation/generation.py`:
```bash
python generation.py -t random -n 10000 -l 30 --U=85 --pd -o output_n10000_l30_u85.dat
```

## Optimization Summary

### Files Modified

| File | Action | Key Changes |
|------|--------|-------------|
| `OMdistance.cpp` | Replace | Raw pointers, removed fake SIMD, SM diagonal zeroed, prefix trimming |
| `LCSdistance.cpp` | Replace | Raw pointers, pre-allocated DP buffers, prefix/suffix trimming, OpenMP |
| `LCPdistance.cpp` | Replace | Raw pointers |
| `CHI2distance.cpp` | Replace | Raw pointers, pre-computed reciprocals, added OpenMP |
| `dp_utils.h` | Replace | Dynamic scheduling, diagonal skip |
| `dist2matrix.cpp` | Replace | Raw pointers, row pointer hoisting |
| `preprocess.cpp` | New | C++ FNV-1a hash dedup (replaces np.unique + seqconc) |
| `module.cpp` | Edit | Added preprocess.cpp binding |
| `get_distance_matrix.py` | Edit | C++ preprocessing, lazy seqdur, removed gc.collect() |

### OpenMP

macOS Conda requires manual setup:
```bash
brew install libomp
export SEQUENZO_ENABLE_OPENMP=1
export CPPFLAGS="-I/opt/homebrew/opt/libomp/include"
export LDFLAGS="-L/opt/homebrew/opt/libomp/lib"
python setup.py build_ext --inplace
pip install -e . --no-build-isolation
```

## How to Reproduce

### Generate datasets
```bash
cd tests/archive_test/xinyi_test/expe_tanat/random-sequences-generation
python generation.py -t random -n 10000 -l 30 --U=85 --pd -o output_n10000_l30_u85.dat
```

### Run benchmarks
```bash
cd tests/archive_test/yapeng_test

# OM
python run_sequenzo.py -d output_n10000_l30_u85
python run_tanat.py -d output_n10000_l30_u85 -m edit

# LCP, LCS, EUCLID
python run_sequenzo_other.py -d output_n10000_l30_u85
python run_tanat_other.py -d output_n10000_l30_u85
```

### Generate plots
```bash
cd plot_runtime
python om_baseline_benchmark.py
python other_baseline_benchmark.py
```

## Test Validation

```bash
cd ~/Desktop/Sequenzo

pytest tests/test_quickstart_integration.py -v -x -k "not test_version"  # 14 passed
```

All optimizations pass existing test suites with zero regressions.
