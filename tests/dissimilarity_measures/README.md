# Dissimilarity measure tests

## Paper-aligned spell OM tests (primary for OMspell)

`test_LCPspell_paper_appendix_sequences_9_10.py` checks illustration Sequences **9--10** (same spell order, duration differences) and **1--2** (different-state single-spell substitution) from the LCPspell paper appendix (`main.tex`, labels `app:om-omspell-illustration` and `app:lcp-variants`):

- **OMspell**: Studer & Ritschard (2016) spell costs with expansion terms `(d_k-1)` and `(d_k+e_\ell-2)` for indel/del and different-state substitution; same-state substitution uses `\lambda|d_k-e_\ell|`.
- **OMspellRS**: full reference-scaled duration (`d_k/\tau`, `(d_i+d_j)/\tau`); Yujian--Bo normalization uses spell-sum delete-all costs for `ml`/`nl`.
- **LCPspell / LCP / LCPmst / LCPprod / OM**: appendix worked distances for both sequence pairs.
- **Seq. 1--2**: highlights OMspell different-state cost `sigma + lambda(d_i+d_j-2)` (raw 12) vs OMspellRS `sigma + lambda(d_i+d_j)/tau` (raw 4).

Run:

```bash
pytest tests/dissimilarity_measures/test_LCPspell_paper_appendix_sequences_9_10.py -v
```

## TraMineR parity tests

`test_dissimilarity_measures_traminer.py` compares many measures to TraMineR reference CSVs generated by `traminer_reference.R` (and related scripts).

**OMspell / OMtspell**: TraMineR comparison tests are **skipped** on purpose. Sequenzo implements the spell-expansion and spell-based normalization conventions described in the LCPspell paper (aligned with Studer & Ritschard 2016 cost definitions), which may differ from `TraMineR::seqdist(..., method="OMspell")` in R. Standard OM, OMstran, LCS, TWED, etc. are still checked against TraMineR where references exist.

To regenerate R references (for non-skipped measures):

```bash
Rscript tests/dissimilarity_measures/traminer_reference.R
```
